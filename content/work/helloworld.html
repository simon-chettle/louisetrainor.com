Helloworld

UX and UI design

Helloworld wanted a brand new interface for their internal systems. The aim was to simplify the UI for new travel agents, but also keep the shortcuts & functionality that exists for current travel agents.

This involved combining multiple complicated systems into one simplified user interface. I was sole designer for this project as well as being heavily involved in all UX decisions.
